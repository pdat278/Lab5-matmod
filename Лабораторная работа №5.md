**II.Решение:**
-	Код в Scilab:

a=0.17;

b=0.046;

c=0.37;

d=0.034;

x0=[11;16];

function dx=syst2(t, x)

    dx(1) = -a*x(1) + b*x(1)*x(2);
    
    dx(2) = c*x(2) - d*x(1)*x(2);
    
endfunction

t0=0;

t = [0: 0.1: 400];

y = ode(x0,t0,t,syst2);

n=size(y,"c");

for i =1:n
    y2(i) = y(2,i);
    
    y1(i) = y(1,i);
    
end

plot(t,y1);

figure;

plot(t,y2);

figure;

plot(y1,y2);

figure;
